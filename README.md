# Frontend

Welcome to **Kati47/frontend**!  
This project showcases advanced, modern frontend architecture and a rich set of features designed for both users and administrators.


- **Modern TypeScript/React codebase:** Robust, scalable, and performant.
- **Great developer experience:** Clean structure, intuitive setup, and easy-to-extend UI components.
- **Responsive & accessible:** Out-of-the-box support for multiple devices and accessibility.
- **Modular features:** All core functionalities are encapsulated and reusable.

## 🛠️ Main Functionalities

- **Product comparison:**  
  Compare products by features, price, technical specs, compatibility, and maintenance. Get functional differences, summaries, best choices for different needs, and tailored buying advice.
- **Room planner:**  
  Interactive tool for configuring a room layout, placing furniture, planning wall vertices, and viewing recommendations. Includes budget planning, 2D/3D switching, and drag-and-drop editing.
- **Admin dashboard:**  
  Rich dashboard with charts and analytics for products, sales, inventory, users, order status, revenue, and more.
- **Product management:**  
  Admin panel to add, edit, delete, and manage products with image and detail forms, bulk updates, and tabbed navigation.
- **Promo codes management:**  
  Create, manage, and apply promo codes with usage limits, applicable products/categories, first-time-only options, and live status toggling.
- **Customer management:**  
  Admin can view, add, edit, and filter customers, assign roles, reset passwords, and review customer history.
- **Order management:**  
  Admin can view, update, and manage orders, track status, export to PDF, and message customers.
- **Landing page:**  
  Conversion-focused home page with dark/light mode, multi-language support, and marketing sections.
- **Modular UI components:**  
  Navigation menus, tabs, cards, badges, forms, calendar picker, dropdowns, buttons, and more—all reusable.
- **User layout:**  
  Header, footer, and chatbot integrated for the user-facing site.

## 🚀 Getting Started

1. **Clone the repository**
    ```sh
    git clone https://github.com/Kati47/frontend.git
    cd frontend
    ```
2. **Install dependencies**
    ```sh
    npm install
    # or
    yarn install
    ```
3. **Start the development server**
    ```sh
    npm start
    # or
    yarn start
    ```
4. Visit [http://localhost:3000](http://localhost:3000)  
   and explore all the amazing features!
